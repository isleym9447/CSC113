Pandas Assignment: Contacts Data Analysis

This assignment focuses on using pandas to manipulate and analyze data from CSV files. You will be working with contacts information stored in CSV format.

1. Loading and Getting Information

  • Load contacts.csv into a DataFrame using pandas, with email as the index.
  • Use info() to show non-null values and column types.
  • Determine row and column count using len() and shape.
  • Display first two and last two rows with head() and tail().

2. Accessing Rows, Columns, and Cells

  • Display the phone column.
  • Show first and last name columns in a DataFrame.
  • Find the phone number for email smsmith@yahoo.com.

3. Filtering and Sorting

  • Sort data by last name, then first name.
  • Show rows where the first name is Dan.

4. Handling Missing Values

  • Display rows with missing phone numbers.

5. Updating Data

  • Set phone to '555-1233' for email tessbijon@bijon.com.
  • Remove the row with email tessbijon@gmail.com.
  • Add a new row with provided data (Stef Smith).

6. Joining Data

  • Join the main DataFrame with state data from contact-states.csv.

This assignment covers various pandas operations, including loading CSV files, data inspection, column and row selection, filtering, sorting, handling missing values, updating data, and joining DataFrames. Each step builds on the previous ones, giving you practice with essential pandas functionality for data manipulation and analysis.
